# Electric-Visions 🚗⚡

A Tesla-inspired electric vehicle landing page built with HTML and CSS.

## Live Demo
[View on Netlify](https://electric-vision.netlify.app/)

## About The Project
Electric-Visions is a front-end clone of the Tesla homepage, 
built as part of an HTML & CSS assignment. The site features 
a hero carousel, a models showcase section, and a footer with 
navigation links. 

## Features
- A navbar including with three parts logo, middle-menu, and the right menu.
- Hero section with CSS-only image carousel (no JavaScript)
- Arrow and dot navigation between slides on both the Hero and Models section.
- Models section with horizontal scroll snap layout
- Responsive design with mobile media queries
- Hover effects on buttons and navigation items
- Sticky navbar

## Built With
- HTML5
- CSS3 (Flexbox, CSS Grid, Scroll Snap, Media Queries)
- Google Fonts (Roboto)

## Sections
- **Navbar** — Fixed navigation with logo and menu links. I added icons for the right menu (support, region language and account)
- **Hero** — Full screen image carousel with 3 slides and the ability to navigate with arrows.
- **Models** — Horizontal scrollable cards for Model 3, Y, S and X.
- **Footer** — I added navigation links for all the content included.

## Images
Car photography sourced from [Unsplash](https://unsplash.com) 
and [Pexels](https://pexels.com).
- I opted for this option as I did not want any Cyper security violation so using unsplash and pexels images helps as they are essentially images taken by photographers and not Tesla's own marketing.
